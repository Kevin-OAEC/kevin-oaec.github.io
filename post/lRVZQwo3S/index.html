CentOS7安装Jenkins

2019-10-08 Jenkins ， Linux

更多内容请查看Jenkins官方文档：https://pkg.jenkins.io/redhat-stable/

下载 Jenkins 的 yum 源

sudo wget -O /etc/yum.repos.d/jenkins.repo https://pkg.jenkins.io/redhat-stable/jenkins.repo
sudo rpm --import https://pkg.jenkins.io/redhat-stable/jenkins.io.key

使用 yum 安装

这里下载过程相当慢慢慢慢慢。。。。。。

yum install jenkins

安装完成之后启动

systemctl start jenkins

输入密码然后继续

启动成功后访问：http://your_ip:8080，会提示密码文件的路径，打开文件复制密码，填入以继续（忘记截图了）

安装建议的插件

又是漫长的等待。。。

创建管理员账户

插件安装完成会要求创建管理员账户，然后就可以使用了
